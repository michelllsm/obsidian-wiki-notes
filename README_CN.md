# 📝 Obsidian AI Notes

> 通过自然语言指令管理 Obsidian 笔记的 AI 工作流

**Obsidian AI Notes** 是一个 AI Skill，让你通过简单指令（如 `-day`、`-note`、`-week`）自动创建、归类和管理结构化的 Obsidian 笔记。

**特点：** 中文为主，英文兼容，根据你的语言自动输出对应语言内容。

---

## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/michelllsm/obsidian-ai-notes.git

# 2. 告诉 AI 助手
-setup

# 3. 跟着引导完成（5步，约5分钟）
# 4. 开始记笔记！
-day
```

## ✨ 核心指令

### 📥 收录与记录

| 指令 | 中文别名 | 什么时候用 | 输出目录 |
|------|---------|-----------|---------|
| `-day` | `-每日笔记` | 一天结束时，记录今天学到的、遇到的问题、明天要做的。AI 自动读取当前对话 + 全局记忆 + 今日已有笔记 | `📅每日笔记/` |
| `-note` | `-主题` | 收录文章、视频、播客——任何值得保留的外部内容，AI 帮你提炼摘要 | `📚主题笔记/` |
| `-input` | `-灵感`, `-输入` | 脑子里闪过一个想法，快速记下来，不需要完整结构 | `📚主题笔记/` |
| `-week` | `-每周笔记` | 周末复盘。AI 自动读取本周所有每日笔记 + 当前对话汇总 | `📆每周笔记/` |

### 📤 输出与跟踪

| 指令 | 中文别名 | 什么时候用 | 输出目录 |
|------|---------|-----------|---------|
| `-output` | `-创作`, `-输出` | 你的原创内容：文章、帖子、分享稿（不与收录混放） | `📝创作输出/` |
| `-task` | `-待办`, `-跟踪` | 需要持续跟踪的任务：有背景、有完成标准、有进度 | `📋待跟踪任务/` |
| `-tool` | `-工具` | 发现了一个好用的工具/技巧，记录下来方便以后查 | `🛠️tools/` |

### 🔄 整理与维护

| 指令 | 中文别名 | 什么时候用 | 输出目录 |
|------|---------|-----------|---------|
| `-ingest` | `-归类`, `-classify` | 收件箱堆了一堆裁剪，让 AI 逐篇判断分类并归档 | 从 `📎Clippings/` 到各分类 |
| `-compile` | `-整合`, `-编译` | 某个主题攒了好几篇笔记，合并成一篇结构化总览 | `📚主题笔记/综合/` |
| `-check` | `-检查` | 定期体检：找出孤立笔记、过期内容、缺失链接 | _(仅输出报告)_ |
| `-update` | `-迭代` | 想到了改进笔记工作流本身的点子 | `💡笔记工作流优化idea/` |

> **提示:** 使用 `-` 前缀加强指令识别。忘记加 `-`？AI 会理解清晰的意图或询问确认。

## 📁 文件夹结构

```
<vault>/
├── 📅每日笔记/                    # -day
├── 📚主题笔记/                    # -note, -input（收录的内容）
│   └── 综合/                      # -compile（AI 编译的主题总览）
├── 📆每周笔记/                    # -week
├── 📋待跟踪任务/                  # -task
├── 📝创作输出/                    # -output（你的原创，不与收录混放）
├── 🛠️tools/                      # -tool
├── 💡笔记工作流优化idea/           # -update
├── 📎Clippings/                   # 收件箱 + 原始全文档案（-归类 扫描此处）
└── obsidian-ai-notes skill配置/   # Skill 文件（可直接修改）
```

**核心设计原则：** 收录内容（`📚主题笔记/`）和原创输出（`📝创作输出/`）永远分开。你始终知道哪些是你写的。

## 🔄 更新与自定义

### 更新

```bash
git pull origin main
```

你的自定义**始终受保护**——AI 智能合并更新，不会覆盖你的修改。详见 SKILL.md § 7。

### 自定义

直接编辑 `obsidian-ai-notes skill配置/` 里的任何文件，或者告诉 AI：

```
帮我修改每日笔记模板，加一个"心情"字段
```

修改立即生效。
